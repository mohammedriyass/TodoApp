# Todo App

A desktop Todo Application built using Java Swing, JDBC and MySQL.

## Features
- Add Tasks
- Update Tasks
- Delete Tasks
- Mark Tasks as Completed
- Persistent Storage using MySQL

## Technologies Used
- Java
- Swing
- JDBC
- MySQL
- MVC Architecture

## Database
Table: tasks

Columns:
- id
- task_name
- status
